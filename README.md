## ◈ Featured Projects

<details>
<summary><b>◉ PrivGuard — AI-Powered Data Redaction & Anonymization Platform</b></summary>

<br/>

> A multi-modal, enterprise-grade anonymization platform built with GDPR and HIPAA compliance at its core. PrivGuard automates the detection and redaction of Personally Identifiable Information (PII) across text documents, PDFs, and audio files — preserving document structure and contextual integrity throughout the process.

<div align="center">

| Attribute | Details |
|---|---|
| **Stack** | Python, React.js, FastAPI, spaCy, WhisperX, PyMuPDF |
| **PII Entity Types** | 10+ including Aadhaar, PAN, email, phone, names, addresses |
| **Compliance** | GDPR / HIPAA aligned privacy controls |
| **Performance** | Coordinate-based PDF redaction, timestamp-accurate audio anonymization |
| **Security** | Utility-preserving pseudonymization with structural integrity |
| **Impact** | Published research on gradational anonymization methodologies |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repository-7C3AED?style=flat-square&logo=github&logoColor=white)](https://github.com/Nishantdubey7/PrivGuard) |

</div>

**Engineering Highlights:**

- Engineered an automated PII detection pipeline using spaCy NER models fine-tuned for Indian document identifiers (Aadhaar, PAN cards), achieving reliable extraction across 10+ sensitive entity categories.
- Implemented coordinate-based PDF redaction using PyMuPDF that preserves document layout, fonts, and visual structure while precisely masking PII at pixel-level coordinates.
- Built timestamp-accurate audio anonymization using WhisperX transcription aligned to audio segments, enabling targeted redaction without disrupting surrounding speech content.
- Co-authored a research publication demonstrating utility-preserving pseudonymization — a technique that anonymizes data while maintaining its analytical and contextual value for downstream processing.

<br/>

</details>
